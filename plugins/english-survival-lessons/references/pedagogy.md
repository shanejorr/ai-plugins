# Pedagogy

All generated content (itinerary, lesson plans, readings, audio scripts, video scripts,
notes handouts) MUST follow this pedagogy. It is research-based and ordered by
priority.

## Priority order

1. **Comprehensible input first.** Most of each lesson is listening / reading they
   can mostly understand: short stories, simple dialogues, narrated scenes. Listening
   before speaking; allow a **silent period** — never force early production.
2. **Task-based practice.** Every lesson ends with one real communicative task
   (role-play: order food, ask a teacher a question, find a classroom).
3. **Light, just-in-time grammar.** Teach a structure only when the day's task needs
   it (question forms, imperatives, simple past). Keep it under ~5 minutes.
4. **Spaced repetition.** Each lesson's media MUST recycle vocabulary from earlier
   lessons. Recycled items appear in reading, audio, and the three videos, not only
   in the warm-up. See "Spaced repetition policy" below.

## Survival / functional focus

- High-frequency words plus fixed phrase frames: "Can I …?", "Where is …?",
  "I don't understand", "How do you say …?".
- Target totals across the whole 20-day course: ~300–500 words and ~100–200 phrases.
- Introduce only ~10–15 new vocabulary items per day; recycle prior items relentlessly.

## Leverage Spanish

Teach Spanish–English cognates explicitly. Provide Spanish glosses for every new
vocabulary item.

**Warn about false friends** (these MUST appear as warnings whenever they show up in
generated material):

- `embarazada` ≠ `embarrassed` (means "pregnant")
- `éxito` ≠ `exit` (means "success")
- `ropa` ≠ `rope` (means "clothes")
- `sopa` ≠ `soap` (means "soup")
- `librería` ≠ `library` (means "bookstore"; English `library` = `biblioteca`)
- `asistir` ≠ `assist` (means "to attend")

## Spaced repetition policy (for daily-lesson)

1. The itinerary specifies which earlier-day vocabulary recycles into each day.
2. The daily-lesson skill builds a coverage set: itinerary's recycled items, topped
   up from oldest earlier-day items if the itinerary list is under 8 items. Minimum
   8 recycled items per day once enough material exists.
3. Each recycled item appears in **at least two** of these five files:
   `reading.md`, `audio-script.md`, `video-script-1.md`, `video-script-2.md`,
   `video-script-3.md`.
4. The `teacher-plan.md` opens with a coverage table showing which item appears
   where; the warm-up drills every item in the table.
5. Recycling must be natural — words used in real prose / dialogue, not bolted on.

## Content constraints for all generated material

- **English level: absolute / low beginner.** Short, simple sentences.
- **Natural American English.** Avoid awkward, stilted, or unidiomatic phrasing.
- Mostly English; use Spanish only for glosses and clarifying brand-new concepts.
- **Age-appropriate for teenagers.** Draw themes from their interests and
  Colombian / Catholic background. Never childish.
- Same lesson and media for both children every day.
- Predictable file naming and folder structure. Zero-pad day numbers to two digits
  (`day-07`, not `day-7`).
- No emojis in any generated material.

## Lower anxiety

- Personalize content to the children's interests (Camilo: soccer, video games,
  drawing, cats, Bible; Luciana: volleyball, reggaeton, fashion). Rotate which kid
  the story features so neither dominates.
- Low-threat practice; frequent small wins.
- Build toward a cumulative can-do checklist (15–20 survival tasks) so the kids see
  measurable progress.
